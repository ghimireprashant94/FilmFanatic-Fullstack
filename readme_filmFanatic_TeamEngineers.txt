Installations:

if you are running the project first time on the machine,please follow below steps:
1. Mongo DB Download:
Log on to the below url and install mongo db
https://www.mongodb.com/download-center
2. Install NodeJS:
Log on to below link and install the node JS for your machine
https://nodejs.org/en/download/

Once installations are complete,
a.Unzip the project file
b.Start terminal
c.Go to the project directory in the terminal and run 'npm -install'

In order to run the project:
a.Go to the dir corresponding to ...Movie_Application\backend-filmfanatic
b.type npm-start
c.Go to the dir corresponding to ...Movie_Application\frontend-filmfanatic
d.type npm-start
e.go to the directory where mongodb is installed in the terminal and start database by using mongod command

After the build is complete, please open chrome or other browser and log on to http://localhost:4200
